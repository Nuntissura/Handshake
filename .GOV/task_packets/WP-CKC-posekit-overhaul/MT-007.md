Scope: implement the first Rust-native Argus product surface as a named MCP facade over the existing native AccessKit/MCP inspection and steering stack.

Target files:
- `src/frontend/handshake_native/src/mcp/argus.rs`
- `src/frontend/handshake_native/src/mcp/mod.rs`

- The implementation is headless and does not foreground Handshake or steal keyboard/mouse/focus.

- backend code
