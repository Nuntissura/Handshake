# Reference Material

This directory is for non-authoritative governance material.

Use it for:

- historical architecture papers
- non-normative kernel/reference implementations
- extracted spec digests
- roadmap audits
- migration studies
- other reference-grade analysis that should not live in active role/shared authority roots

Rules:

- Files here are not live workflow authority.
- They may preserve historical paths/tokens for evidence value.
- New active workflow guidance belongs under `/.GOV/roles/`, `/.GOV/roles_shared/`, `/.GOV/templates/`, or `/.GOV/docs/` instead.

Subfolders in use:

- `master_spec/`
  - extracted master-spec studies, digests, and roadmap audits
- `kernel/`
  - non-normative concrete reference implementations of the governance kernel
