# Reference Material

This directory holds non-authoritative governance reference material and archaeology.

Use it for:

- `PAST_WORK_INDEX.md` as the top-level archaeology index
- `legacy/` for retired artifacts kept for evidence value
- `research_and_papers/` for synthesis papers and research notes

Rules:

- Files here are not live workflow authority.
- Historical paths or tokens may appear for evidence value only.
- Active workflow guidance belongs under `/.GOV/roles/`, `/.GOV/roles_shared/`, `/.GOV/templates/`, or `/.GOV/docs/`.

Subfolders in use:

- `legacy/validator/`
  - retired validator-state artifacts retained for compatibility archaeology
- `legacy/orchestrator/`
  - archived non-authoritative Orchestrator support docs retained for historical reference
- `research_and_papers/`
  - non-normative synthesis and research material
