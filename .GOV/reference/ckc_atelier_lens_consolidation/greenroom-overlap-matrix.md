# Atelier/Lens And CKC Overlap Matrix

This matrix records CKC as an evolved sibling source for Atelier/Lens, not as optional inspiration. Rows must be folded into the consolidation packet unless marked as a separate dependency, deferred feature family, conflict, or operator-decision-needed.

SQLite remains forbidden in Handshake in every form: runtime, tests, fixtures, mocks, examples, fallbacks, cache, compatibility adapters, temporary harnesses, imports, exports, or product paths.
